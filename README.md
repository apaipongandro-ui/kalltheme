# 🎨 Nebula Pro Theme - Pterodactyl Panel

Dark purple theme with **Protect Manager** and **Canva-style design** for Pterodactyl Panel.

## ✨ Features
- Dark purple gradient + glassmorphism
- Animated nebula particles
- Built-in security (XSS, CSRF, SQLi, Brute Force)
- **Protect Manager** (only accessible by User ID 1)
- Anti-Intip Server
- Hide Buttons (only show Server & User)
- Mode Stealth (hide resource usage)
- Dashboard Lock
- Mobile responsive

## 🚀 Installation

```bash
git clone https://github.com/apapiongardo-ui/kalltheme
cd kalltheme
chmod +x install.sh
./install.sh
